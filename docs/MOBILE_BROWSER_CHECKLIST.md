## 01 — 首页 / Home

### 视觉检查
- [ ] 页面标题 "MiniMax 音乐创作台" 无溢出
- [ ] 副标题/描述文字不超出视口
- [ ] 导航栏（创作 / 作品 / 模板 / 设置 / 文档）无溢出
- [ ] CTA 按钮明显可点击
- [ ] 无横向滚动条

### 内容检查
- [ ] 产品名称显示正确
- [ ] 后端状态指示器可见（如"MMX CLI"）
- [ ] 无 API key 泄露

## 02 — 创作台 / Studio

### 模式选择
- [ ] 四个模式 Tab（纯音乐 / 自动成歌 / 歌词成歌 / 参考改编）横向排列
- [ ] Tab 切换流畅
- [ ] 当前选中模式高亮
- [ ] Tab 在 375px 宽度下不拥挤

### 表单区域
- [ ] 音乐描述 textarea 可见，易于点击
- [ ] 风格标签（安静/明亮/梦幻/电影感/电子/钢琴/吉他/Lo-fi）横向滚动或换行
- [ ] 标签点击区域 >= 44px
- [ ] 高级设置展开/收起正常

### 生成进度 UI（Phase 2K 新增）
- [ ] 进度消息区域在 textarea 和生成按钮之间
- [ ] "正在提交任务…" / "正在调用 MMX CLI…" / "生成中…" 消息清晰
- [ ] elapsed time 计数可读（格式：0:03 / 0:15）
- [ ] 进度条不为零时可见

### 生成按钮
- [ ] "生成音乐" / "生成纯音乐" 按钮明显
- [ ] 生成中按钮变为 "生成中…" 并 disabled
- [ ] 防重复点击生效

### 播放器卡片
- [ ] 播放器卡片显示在按钮下方或侧边
- [ ] 播放按钮 >= 44px
- [ ] 时长显示正确（不为 "???" 或 "0:00"）
- [ ] 加载中显示 "读取中"
- [ ] 下载 MP3 按钮可见可点击
- [ ] 标题为中文（"周末加班听的音乐" 等）
- [ ] 无横向溢出

### 最近作品列表
- [ ] 作品列表在页面底部或侧边
- [ ] 卡片不过于拥挤

## 06 — 通用移动端问题排查

### 布局
- [ ] 无横向滚动（`overflow-x: hidden` 或无溢出）
- [ ] 字体大小适合手机阅读（不小于 13px）
- [ ] 卡片圆角 20px 左右
- [ ] 按钮圆角 999px（胶囊形）

### 触控
- [ ] 所有可点击元素 >= 44×44px
- [ ] 标签之间间距适当
- [ ] 无 hover 依赖（hover 仅作增强，不作功能必需）

### 性能
- [ ] 首屏 3 秒内可见内容
- [ ] 无明显布局跳动（CLS）
