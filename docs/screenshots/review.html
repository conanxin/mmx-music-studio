设计系统

色彩

背景 #090A0C 面板 #121419 次面板 #181B21 边框 #262A33 强调 #B8FF6A 辅助 #8A7CFF 错误 #FF6B6B

字体

系统字体栈（SF Pro / PingFang / Noto Sans SC / Roboto），无外部字体依赖。

圆角

卡片圆角 20px，按钮圆角 999px（胶囊形），输入框 12px。

间距

4px 为基础单位，常用：16px（space-4）、20px（space-5）、24px（space-6）。

按钮触控区域

所有按钮最小高度 44px，符合移动端无障碍标准。

Phase 2A 验收清单

  • ✓ packages/core 平台无关（无 React/DOM/BOM 依赖）
  • ✓ 四种模式：instrumental / auto / lyrics / cover-url
  • ✓ instrumental → is_instrumental=true payload
  • ✓ auto → lyrics_optimizer=true payload
  • ✓ lyrics → payload.lyrics 字段
  • ✓ cover-url → payload.audio_url 字段
  • ✓ cover-file → needsUpload 标记（无 base64 时）
  • ✓ validateMusicInput 完整校验，中文错误文案
  • ✓ createMockTrack / createMockJob / advanceMockJob
  • ✓ UI 点击生成 → validate → mock job → success track
  • ✓ 错误显示（红色框），最近作品列表自动追加（最多 3 条）
  • ✓ npm run typecheck 通过
  • ✓ npm run build 通过（204KB JS）
  • ✓ smoke test 全部通过
  • ✓ 中文界面，简洁设计，移动端优先
  • ✓ 微信小程序准备：core 不依赖 Web API
  • ✓ 无 API key 读取、输出、保存
  • ✓ .env.example 只含占位符
