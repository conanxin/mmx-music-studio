# MiniMax 音乐创作台 — UI 设计规范

## 1. 整体风格

- **中文界面**，全产品线中文文案
- **简洁**，轻量，音乐创作工具感
- 少装饰、少霓虹、少复杂面板
- 保留高级感，但比暗色复杂界面更干净
- **移动端优先**，桌面端舒适双栏布局

## 2. 视觉方向

### 色彩系统

```
背景（最深）：     #090A0C   → 页面最底层背景
主面板：           #121419   → 卡片、容器背景
次面板：           #181B21   → 输入框、下拉框背景
主文字：           #F4F1EA   → 标题、重要文字（暖米白）
次文字：           #9BA1AA   → 说明文字、占位符
边框：             #262A33   → 分割线、输入框边框
强调色（主按钮）：  #B8FF6A   → 主按钮背景（柔和绿）
辅助色：           #8A7CFF   → 次要强调、标签（薰衣草紫）
错误色：           #FF6B6B   → 错误提示、警告
成功色：           #6BFFB8   → 成功提示（薄荷绿）
```

### 字体

- 主字体：`Inter`（Google Fonts），weight: 400 / 500 / 600 / 700
- 中文字体回退：`"PingFang SC", "Microsoft YaHei", sans-serif`
- 代码字体：`"JetBrains Mono", "Fira Code", monospace`

### 间距系统

- 基础单位：4px
- 常用间距：8px / 12px / 16px / 24px / 32px / 48px / 64px
- 大量留白，避免拥挤

### 圆角

- 卡片圆角：20px
- 按钮圆角：999px（全圆角药丸形）
- 输入框圆角：12px
- 标签圆角：999px

### 边框

- 减少边框数量，优先用背景色差区分层次
- 必须有边框时：`1px solid #262A33`

## 8. 组件规范

### 按钮

- **主按钮**：背景 `#B8FF6A`，文字 `#090A0C`，圆角 `999px`，高度 `48px`
- **次按钮**：背景 `transparent`，边框 `1px solid #262A33`，文字 `#F4F1EA`
- **危险按钮**：背景 `#FF6B6B`，文字 `#FFFFFF`
- **禁用态**：透明度 `40%`，不可点击

### 输入框

- 背景：`#181B21`
- 边框：`1px solid #262A33`
- 圆角：`12px`
- 聚焦态边框：`#B8FF6A`
- 内边距：`12px 16px`
- 字体：`16px`（防止 iOS 缩放）

### 标签（Tag）

- 背景：`#181B21`
- 边框：`1px solid #262A33`
- 选中态背景：`#B8FF6A`，文字：`#090A0C`
- 圆角：`999px`
- 内边距：`6px 14px`

### 卡片

- 背景：`#121419`
- 圆角：`20px`
- 内边距：`24px`
- 边框：无（靠背景色差区分）

## 9. 动画规范

- 过渡时间：`200ms ease`
- 生成中加载：简洁的点状脉冲，不使用复杂 spinner
- 播放器播放中波形：CSS animation 模拟微微跳动
- 不使用过度复杂的动画，避免性能问题

## 10. 图标

- 使用 Lucide Icons（轻量、一致性好）
- 通过 CDN 或 npm 引入
- 图标颜色跟随文字颜色

## 11. 无障碍规范

- 所有交互元素有 `focus` 样式
- 图片有 `alt` 文案
- 颜色对比度符合 WCAG AA 标准
- 表单标签清晰
- 按钮点击区域至少 `44px × 44px`（移动端）
