# Feedback Checklist: Public Lite BYOK

请按第一次试用的真实感受回答。简短回答即可。

1. 第一次打开页面时，你是否知道下一步该怎么用？
2. MiniMax API Key 输入说明是否清楚？
3. 一键生成模板是否有帮助？你会想要哪些模板？
4. 生成等待时间是否可以接受？
5. 如果生成失败，失败提示是否清楚、可行动？
6. 作品库是否容易理解？你是否知道生成后去哪里找作品？
7. 播放和下载是否顺畅？
8. 你最希望下一步增加什么？

补充信息：

- 这是 5 人内轻量公开 BYOK demo。
- 你需要使用自己的 MiniMax API Key。
- 本站不保存 API Key。
- 生成任务会排队执行。
- 超过 5 个活跃用户时，生成会暂时暂停，但页面和作品库仍可浏览。
