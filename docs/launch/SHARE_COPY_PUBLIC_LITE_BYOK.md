## X / Twitter

I shipped a small Public-Lite BYOK music generation demo.

It lets a few trusted users try AI music generation with their own MiniMax API Key. Jobs are queued, the demo is capped at 5 active users, and the site does not persist API Keys. Generated tracks can be played, downloaded, and viewed in the Library.

This is a personal/demo project, not a broad public launch. Feedback welcome.

Try it: https://music.conanxin.com/studio

## 中文技术社区版

我把 `mmx-music-studio` 收口成了一个可以小范围分享的 Public-Lite BYOK 音乐生成 demo。

当前版本是 `v0.5.0-public-lite-byok`：

- 5 人内轻量公开试用，不是大规模 public launch
- 用户使用自己的 MiniMax API Key
- 本站不保存 API Key
- 生成任务单 worker 排队执行
- 生成结果可播放、下载，并在 Library 查看
- Studio 页面提供三步说明和一键 prompt 模板

这个版本主要想验证首屏说明、BYOK 输入、排队等待、播放下载和作品库体验是否顺畅。

试用入口：https://music.conanxin.com/studio
